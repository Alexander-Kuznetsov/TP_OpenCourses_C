с) Max-норма матрицы, находящейся во входном файле
1) Не разреженный – в первой строке файла записано количество строк матрицы; во второй -
количество столбцов матрицы; в последующих строках файла записаны элементы строк
самой матрицы (по одной строке матрицы на одной строке файла)
1) Двумерный массив вещественных чисел

О программе: 
Программа может работать как с обычной матрицей, так и с разреженной матрицей
Однако, даже при разреженной матрице все сохраняется в двумерный массив.

Пример входных данных:
______
2
2
1 2
3 4
______
2
2
2
0 0 1
1 1 2
______
Если при обработке разреженной матрицы в файле больше ненулевых элементов, чем 
указано во входном параметре, то они учитываться не будут.
Ситуация:
______
2
2
2
0 0 1
0 0 2
0 1 3
______
Будет обработана

Если во входном файле матрица больше, чем указано в параметрах, то такая матрица обрабатываться не будет.

Перед параметрами допускается табуляция, т.е.
______
   2
2
1 2
3 4
______

Допускается ввод матрицы построчно, т.е.
______
2
2
1 2 3 4
______
Или
______
2
2
2
0 0 1 0 1 2
______
